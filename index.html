PBSU T2 Flexi checker

Upload only the finalized RMU Flexi roster. This page will auto-detect the date from the finalized filename, read the matching date tab from your Google Sheet, check column G cancellation status, then export every valid officer from the Google Sheet date tab. Officers found in finalized CSSU T2 use finalized roster details; officers not found in finalized are still included using Google Sheet ID/name/shift. Random/non-officer rows are ignored.

1) Upload Finalized RMU Flexi Roster

Example filename: RMU FLEXI ROSTER 070526 (FINALISED).xlsx. Website will convert 070526 to Google Sheet tab 07.05.2026, and auto-try tabs with hidden/leading spaces such as   07.05.2026.

2) Connected Google Sheet

The Google Sheet must be shared as Anyone with the link can view. Otherwise a standalone HTML page cannot read it.

Check settings

If the date tab has hidden spaces, this version will auto-try the space version. If Google blocks the read, open the Google Sheet → Share → General access → Anyone with the link → Viewer. If your company does not allow public link sharing, use Google Apps Script Web App / Google Sheets API OAuth instead.
